## Описание проекта и его функциональности

Данный проект - это телеграм бот, работающий с Google Calendar API, написанный для Облака КРОК. 
Основной функционал:
1. Возвращение текущего или следующего события
2. Поиск по дате

## Технологии
* Node.js
